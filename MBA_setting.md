## 設定

* Caps Lock -> Ctrl
* ライブ変換OFF
* マウススピード早く
* ネットアカウント設定

## ショートカット

* Launch 表示　Command + L
* Finder+検索　Command + R
* スポットライト検索　Command + Space
* デスクトップ表示　ctrl + ↓
* 入力切り替え　Option + `
